ShieldAI - AI-Powered Phishing Detection System
A real-time desktop application that protects users from phishing attacks using machine learning, OCR screen monitoring, and VirusTotal API verification.

🚀 Features
Real-Time Protection
Screen Monitoring: Continuously scans your screen for suspicious URLs using OCR
Instant Alerts: Dynamic Island-style popup notifications for detected threats
Hybrid Verification: Combines local AI models with VirusTotal API for accurate detection

Manual Analysis
URL Deep Scan: Analyze individual URLs with detailed risk scoring
SHAP Explanations: Understand why the AI made its decision
API Integration: Cross-verify results with VirusTotal

Visual Analytics
18+ Professional Graphs: Performance metrics, feature importance, confusion matrices, and more
Live Event Stream: Terminal-style log with color-coded threat indicators
Dark Theme UI: Modern macOS-inspired design

📋 System Requirements
OS: Windows 10/11 (Linux/Mac with modifications)
Python: 3.8 or higher

Tesseract OCR: Installed at C:\Program Files\Tesseract-OCR\tesseract.exe
